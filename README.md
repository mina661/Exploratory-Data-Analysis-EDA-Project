#  Hotel Booking EDA Project

## Overview
Exploratory Data Analysis on a hotel booking dataset
to understand why customers cancel their bookings.

## Dataset
- 119,390 hotel bookings
- 10 features including hotel type, market segment, and deposit type

## Key Findings
- 37% of all bookings are canceled
- City Hotel has higher cancellation rate (41.7%) vs Resort Hotel (27.7%)
- Customers who book earlier tend to cancel more
- Groups have the highest cancellation rate (61%)
- Direct bookings have the lowest cancellation rate (15.3%)
- Customers with more special requests cancel less

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- SciPy
